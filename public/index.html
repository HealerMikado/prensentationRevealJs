Reveal js

Un framework HTML pour faire vos présentations

Pourquoi utiliser reaveal js ?

Problèmes récurrents

  • Absence de powerpoint/LibreOffice sur le poste client
  • Présentation déformées à cause de la config du poste client
  • Mince ! J'ai oublié ma clef ...
  • Je trouve les Beamers moche

Avantage de reveal.js

  • Simplicité
  • Multiplateforme (tous sauf IE>9 ou navigateurs sans js)
  • Accessible depuis le web
  • C'est quand même plus simple et beau qu'un Beamer

Installation

  1. Télécharger la dernière version
  2. Unzip
  3. Modifier le index.html
  4. Enjoy

Mais encore

  • Beaucoup d'options déjà fournies
    • Theme
    • Fragment
    • Colorisation du code ...
    • etc ...
  • Mais vous pouvez faire les votres ! -> CSS/JS
  • Avec Emmet (cf il y a longtemps) écrire votre présentation est rapide
  • Et on peut la mettre en ligne sur github/gitlab

La mise en ligne

Step 1 : configurer sa présentation

  1. Mettre la présentation dans un dossier public
  2. A la racine du projet rajouter un fichier .gitlab-ci.yml

.gitlab-ci.yml


pages:
  stage: deploy
  tags:
  - shell
  script:
  - echo 'Deploy to Gitlab pages'
  artifacts:
    paths:
    - public
  only:
  - master
